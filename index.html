川瀬さん用 ハンズオン
フォーム営業ツール 試運転

上から順に「コピー」ボタンを押して貼るだけで試運転が始められる形にしました。 考えることは基本ゼロです。詰まったら青山さんに電話（初回は 20〜30 分横で見ています）。

この 3 点だけ最初に覚えてください
  1. コピーボタンを押す → ターミナルかAI アシスタントに貼る（それぞれ書いてあります）
  2. 途中で AI が質問してきたら、そのまま答える（会社名や URL）
  3. ブラウザに管理画面が出たら、そこで 「承認して自動送信」 ボタンを押すだけ

1ターミナルを開く

Mac 画面右上の虫眼鏡（Spotlight）を押して、下の 6 文字を入力 → Enter。

ターミナル
黒い画面が開けば成功です。以降はここに「コピペ」していきます。

2準備の下ごしらえ（Homebrew を入れる）

初回だけ必要な下ごしらえです。青山さんが横で見ています。パスワードを聞かれたら Mac のログインパスワードを入力してください。

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
数分かかります。最後に Installation successful! と出れば成功。もし途中で止まったら青山さんに電話。

3AI アシスタントの土台を入れる

下の 1 行をターミナルにコピペ。数十秒で完了します。

brew install node git
node と git という部品が入ります。既に入っていれば「already installed」と出るだけで問題なしです。

4AI アシスタント本体を入れる

これでいよいよ AI アシスタントが Mac に入ります。

npm install -g @anthropic-ai/claude-code
1〜2 分。エラーが出たら青山さんに電話。

5AI アシスタントを起動してログイン

下を貼って Enter → 初回だけブラウザが開いてログイン画面が出ます。青山さんが横で認証します。

claude
ログインが済むと、ターミナルの中に AI に話しかける入力欄 が出ます。以降はここに「決まり文句」を貼っていきます。
この場面で一旦区切って青山さんへ
今日はここまでで OK。明日から実運用スタートです。

6【明日 STEP 1】試運転ツールを取り込む

ターミナルに戻って（AI アシスタントを一度 Ctrl+C で閉じてから）下を貼る。

cd ~/Documents && git clone https://github.com/gift-org/form-sales.git && cd form-sales && claude
試運転ツールが Mac に取り込まれ、その場所で AI アシスタントが開きます。

7【明日 STEP 2】AI にセットアップしてもらう

ここが一番のポイントこれを AI に貼るだけで、必要なもの全部インストールしてくれます。

試運転できるようにセットアップして
5〜10 分ほど。途中で Mac のログインパスワードを求められたら入れる。
完了すると AI が「次は会社の情報を教えてください」と聞いてきます。

8【明日 STEP 3】会社の情報を答える

AI が順番に質問してくるので、そのまま答えるだけ。会社名は？ → 答える → 担当者名は？ → 答える。この繰り返し。

聞かれる項目（覚えなくて OK。順番に聞かれます）：

  • 会社名
  • 担当者名
  • 担当者名のフリガナ（カタカナ）
  • メールアドレス（共用の contact@gift-original.jp）
  • 電話番号
  • 自社の URL
  • 初期文面 1 本（AI が例を出してくれるので、そのままで OK）
全部答え終わると AI が「セットアップ完了しました」と報告してくれます。

9【明日 STEP 4】試送する会社の URL を教える

これから会社のフォーム URL を教えるので、試送リストに追加して。
AI が URL を投入して、リストに何件入ったか報告します。

10【明日 STEP 5】本番の前にリハーサル

本物に送る前に、練習用フォームに 1 通だけ流して動作確認します。

本番の前にリハーサルして

11【明日 STEP 6】管理画面を開く

ここで一度ブラウザ側に移ります。

管理画面をブラウザで開いて
Safari か Chrome で管理画面が開きます。左メニューから 「確認待ち」を開いてください。

12【明日 STEP 7】1 件ずつ内容を見て「承認して自動送信」

ここが川瀬さんの一番大事な仕事。

  1. 「確認待ち」に並んでいる 1 件目をクリック（詳細画面が開きます）
  2. 会社名・貼付け項目・文面本文が全部見えます。ざっと目を通して問題なさそうなら
  3. 青い 「承認して自動送信」ボタンを押す
  4. 確認ダイアログ「〇〇株式会社のフォームに送信します。よろしいですか？」→ OK
  5. 「送信中...」→ 数十秒〜1 分で「送信済み」に変わります
  6. 次の 1 件へ。同じ流れ
大事：承認するのは川瀬さん本人だけ
AI は勝手に押しません。川瀬さんが目で見て「これでいい」と思ったものだけ送ります。

13成績を確認したいとき

今日の成績を教えて
AI が「送信 N 件・返信 N 件・アポ N 件」とまとめて教えてくれます。

14返信が来たらチェック

相手企業から返信が来ると、共用アドレス経由で青山さん・川瀬さん両方の個人メールに届きます。管理画面の 「返信」タブでも見られます。

  • 返信を開いて、種別（前向き／質問／お断り／自動応答）を選ぶ
  • 前向きな返信は 「アポ獲得としてマーク」ボタン
  • ダッシュボードの「アポ数」に反映される
