フォーム営業ツール 試運転

Mac に何かを入れる作業は一切ありません。 VPN で会社の中に入って、URL を開いてログインするだけです。

おおよその流れ
  1. Mac の VPN を接続する
  2. ブラウザで URL を開く
  3. ログイン → 使う

1VPN を接続する

コラボリストの時と同じ VPN 設定でそのまま繋がります。Mac のメニューバーから VPN を「接続」にしてください。

接続できたか分からないときは、下の URL が開けるかで判断できます（開ければ VPN OK）。

2URL を開く

ブラウザ（Safari か Chrome）で下の URL を開きます。

URL
http://192.168.1.202:3000/
URL は Slack DM にも貼ってあります。ブックマーク推奨。

3ログイン

ID とパスワードは Slack の青山さんとの DM に届いています。そちらを参照してください（このページには載せていません）。

初回はブラウザに保存しておくと次回から自動で入ります。

4初回だけ：会社情報を入れる

ログインすると管理画面が開きます。左のメニューから 「送信元」 を選んで、以下 6 項目を入力して保存。

  • 会社名
  • 担当者名
  • 担当者名のフリガナ（カタカナ）
  • メールアドレス（共用アドレス）
  • 電話番号
  • 自社の URL
1 回入力すれば次回以降は不要です。あとから変更したいときも同じ「送信元」画面から。

5毎日の業務（送る）

青山さんから当日、実在の会社の URL がいくつか渡されます。

  1. 左メニューの 「確認待ち」 を開く
  2. 1 件目をクリックして詳細を開く
  3. 会社名・貼付け項目・文面本文が見えるので、目を通す
  4. 問題なさそうなら青い 「承認して自動送信」 ボタン
  5. 確認ダイアログ「〇〇株式会社のフォームに送信します。よろしいですか？」→ OK
  6. 「送信中...」→ 数十秒〜1 分で「送信済み」に変わる
  7. 次の 1 件へ、同じ流れ
送信ボタンを押すのは川瀬さん本人だけ
自動で勝手に送られることはありません。目で見て「これでいい」と思ったものだけが送られます。

6成績を見る・返信を捌く

成績

左メニューの 「ダッシュボード」（トップ画面）で、送信数・返信数・アポ数を確認。

返信

相手企業から返信が来ると、左メニューの 「返信」 に自動で並びます。

  • 返信を開いて、種別（前向き／質問／お断り／自動応答）を選ぶ
  • 前向きな返信は 「アポ獲得」 ボタン → ダッシュボードのアポ数に反映

7止めたいとき

画面の 右上に赤い「全停止」ボタン が常にあります。押せば即停止。何か違和感を感じたら迷わず押して大丈夫です。

8困ったとき

  • エラーが出た・見えない・動かない → Slack か電話で青山さんへ
